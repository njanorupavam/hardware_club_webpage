# Hardware Club Website

A professional, futuristic website for the Hardware Club at IEDC Sahrdaya.

## Features
- **Design**: Cyberpunk/High-Tech aesthetic with Neon accents.
- **Tech Stack**: HTML5, CSS3, JavaScript (Vanilla).
- **Animations**: GSAP (GreenSock Animation Platform) for smooth scroll and entrance effects.
- **Interactivity**: Custom cursor, 3D card flips, Glitch text effects.

## How to Run
Simply open `index.html` in any modern web browser.

## Content
- **Home**: Glitch title effect.
- **Goals**: Three pillars of importance (Innovation, Hands-on, Future Ready).
- **Team**: Leadership cards with hover flip effects.
- **Contact**: Direct email link.
